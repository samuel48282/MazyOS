## 1. Post Instagram/Facebook (1080x1350 - Portrait)

### Estrutura

```
┌─────────────────────────────┐
│  FUNDO: Gradiente           │
│  Dourado → Marrom Quente    │
│  (vertical, leve textura)   │
│                             │
│     📖 [Versículo]          │
│     "Texto em branco        │
│      grande, impactante"    │
│                             │
│     ✨ Luz divina           │
│     (efeito glow)           │
│                             │
│  [Reflexão resumida]        │
│  texto menor, creme         │
│                             │
│  ────────────────────       │
│  🙏 O Bom Pastor           │
│     Logo + Data             │
└─────────────────────────────┘
```

### Especificações de Design

**Fundo:**
- Gradiente 45°: #D4A574 (topo) → #7A5D4F (base)
- Adicionar textura sutil (ruído 3-5%)
- Overlay leve de transparência (opcional)

**Versículo:**
- Font: Montserrat Bold ou Inter Bold
- Tamanho: 48-56pt
- Cor: Branco (#FFFFFF)
- Alinhamento: Centro
- Linha de espaço: 1.3-1.5
- Sombra: Sutil (blur 3px, offset 1px)

**Ícones/Emojis:**
- 📖 (bíblia antes do versículo)
- ✨ (esperança, pode ter efeito brilho)
- 🙏 (no rodapé, antes do logo)

**Reflexão Resumida:**
- Font: Open Sans Regular, 18-22pt
- Cor: #F5F1ED (creme)
- Max 3 linhas
- Alinhamento: Centro

**Logo + Rodapé:**
- Logo "O Bom Pastor" (preto ou dourado)
- Data: DD.MM.YYYY
- Altura total rodapé: ~80px
- Fundo: Cor base (marrom)

### Exemplo de Prompt para Design

```
Crie um post Instagram com design cinematográfico para o versículo "João 3:16".

Especificações:
- Fundo: Gradiente suave de dourado quente (#D4A574) para marrom (#7A5D4F)
- Versículo em branco grande (Montserrat Bold 54pt), centrado
- Ícone 📖 acima, ✨ ao redor (com efeito glow)
- Texto de reflexão em creme (#F5F1ED), máximo 3 linhas
- Rodapé: Logo "O Bom Pastor" + data
- Estilo: Cinematográfico, emocional, espiritual
- Sem texto genérico — usar exatamente: "João 3:16"
```

## 2. YouTube Shorts (1080x1920 - Vertical 9:16, 60-90s)

### Estrutura de Cenas

```
CENA 1 (0-5s): ABERTURA ÉPICA
├─ Paisagem cinematográfica
│  (campo, pastor com ovelhas, montanha, luz divina)
├─ Som: Música instrumental orante
└─ Efeito: Transição suave (fade ou pan)

CENA 2 (5-20s): VERSÍCULO
├─ Fundo: Gradiente dourado/marrom
├─ Texto: Versículo em grande tipografia
├─ Animação: Palavra por palavra (fade-in)
├─ Som: Música continua
└─ Efeito: Partículas douradas caindo

CENA 3 (20-40s): REFLEXÃO NARRADA
├─ Visual: Cenas de natureza/pastor/paz
├─ Narração: Voz calma e profunda
│  (reflexão emocional do versículo)
├─ Som: Música instrumental + narração
└─ Efeito: Sobreposição de texto em highlights

CENA 4 (40-60s): CTA + ENCERRAMENTO
├─ Visual: Logo "O Bom Pastor"
├─ Texto: "Compartilhe este versículo"
├─ Som: Música sobe de volume
└─ Efeito: Fade para branco/dourado
```

### Especificações Técnicas

**Resolução:** 1080x1920 (9:16)
**Duração:** 60-90 segundos
**Frame Rate:** 30fps
**Codec:** H.264

**Tipografia (em vídeo):**
- Títulos: Montserrat Bold 60-72pt (branco)
- Corpo: Open Sans 32-40pt (creme)
- Sombra: Sim (2-3px blur, offset 1-2px)

**Som:**
- Música instrumental orante (120-140 BPM)
- Narração: Voz profunda e calma (homem ou mulher)
- Volume: Narração -6db da música
- Fade in/out: 0.5s

**Efeitos Visuais:**
- Partículas douradas (opcional, use com moderação)
- Luz divina/glow (efeito de raios de luz)
- Transições: Fade, corte seco, pan suave
- Intensidade: Dramático mas não sensacionalista

### Exemplo de Briefing para Videógrafo

```
Crie um YouTube Short de 60s sobre João 3:16

Cena 1 (0-5s): Paisagem épica de pastor cuidando de rebanho ao entardecer
Cena 2 (5-20s): Verso "João 3:16" em tipografia cinematográfica, fundo dourado
Cena 3 (20-40s): Narração: "Você já foi amado sem pedir nada em troca?..."
Cena 4 (40-60s): Logo "O Bom Pastor" com CTA "Compartilhe este versículo"

Tone: Emocional, espiritual, esperançoso
Cores: Dourado quente, branco, marrom
Música: Instrumental orante (sugestão: Hymns/Gospel instrumental)
```

## 3. TikTok/Reels (9:16, 15-60s)

### Estrutura Rápida

```
FRAMES 0-3s: GANCHO VISUAL
├─ Versículo em animação (palavra por palavra)
├─ Efeito: Slide, zoom suave, fade
└─ Som: Batida de música viral cristã

FRAMES 3-10s: REFLEXÃO (Visual + Narração rápida)
├─ Cortes rápidos (1-2s cada)
├─ Versículo reaparece periodicamente
└─ Transições: Rápidas mas não caóticas

FRAMES 10-15s: CTA
├─ "Compartilhe" em grande texto
├─ Logo O Bom Pastor
└─ Sound: Música pico
```

### Especificações

**Resolução:** 1080x1920 (9:16)
**Duração:** 15-60s (comece com 30s)
**Estilo:** Fast-paced, trendy, emocional
**Versículo na tela:** 70% da altura

**Música:**
- Trending cristã/gospel instrumental
- Energética mas não frenética
- Deve acompanhar o ritmo

**Efeitos:**
- Transições rápidas (0.2-0.4s)
- Zoom in/out em palavras-chave
- Filtro warmth leve (+10-15% saturação de ouro)
- Partículas (opcional, baixa intensidade)

## 4. Carousel (5-7 slides, 1080x1350 cada)

### Estrutura 5-Slide Padrão

```
SLIDE 1: CAPA
├─ Grande imagem de pastor/ovelhas/luz divina
├─ Versículo sobreposto em branco
├─ Fundo: Gradiente dourado
└─ CTA: "Deslize para reflexão" (setas)

SLIDE 2: CONTEXTO BÍBLICO
├─ Fundo: Branco creme (#F5F1ED)
├─ Título: "Contexto" em dourado
├─ Texto: Contexto histórico (100-150 palavras)
└─ Ícone: 📖

SLIDE 3: REFLEXÃO EMOCIONAL
├─ Fundo: Gradiente suave
├─ Título: "O que isso significa para você" em dourado
├─ Texto: Reflexão (150-200 palavras)
└─ Ícone: 💭

SLIDE 4: APLICAÇÃO PRÁTICA
├─ Fundo: Branco/creme
├─ Título: "Aplicação Prática" em dourado
├─ Ação: Descrição clara e viável
└─ Ícone: ✨

SLIDE 5: CTA + COMUNIDADE
├─ Fundo: Gradiente dourado → marrom
├─ Título: "Compartilhe" em branco grande
├─ Texto: "Com alguém que precisa ouvir hoje"
├─ Logo O Bom Pastor
└─ Ícone: 🙏
```

### Especificações de Design

**Slides alternados:**
- Slide par (2,4): Fundo branco/creme
- Slide ímpar (1,3,5): Fundo gradiente dourado

**Tipografia:**
- Títulos: Montserrat Bold 44-52pt
- Corpo: Open Sans 20-24pt
- Line height: 1.4-1.6

**Cores de Texto:**
- Slide escura: Branco (#FFFFFF)
- Slide clara: Preto/Dourado (#1A1A1A ou #D4A574)

**Espaçamento:**
- Margins: 60px em cada lado
- Padding: 40px entre elementos
- Gap: 20px entre linhas de texto

## 5. Vídeo Longo (2-5 minutos)

### Estrutura Sugerida

```
INTRO (0-30s):
├─ Abertura cinematográfica (paisagem)
├─ Versículo em tela
└─ "Vamos explorar juntos" (narração)

CONTEXTO (30s-1:30min):
├─ Narração sobre história do versículo
├─ Visuals: Contexto histórico, arte, natureza
└─ Música: Instrumental orante

REFLEXÃO (1:30-3:00min):
├─ Narração profunda sobre significado
├─ Visuals: Pessoas em momentos de fé
├─ Texto destacado em tela
└─ Emocional, não pergunta retórica

APLICAÇÃO (3:00-4:00min):
├─ "Como viver isso hoje"
├─ Exemplos práticos
├─ Encorajamento
└─ Esperança

ENCERRAMENTO (4:00-5:00min):
├─ Logo + chamado à ação
├─ "Compartilhe com alguém"
├─ Música sobe
└─ "O Bom Pastor" em branco
```

## Checklist de Qualidade Visual

Para cada asset, verificar:

- [ ] Cores no padrão (dourado, branco, marrom, verde)
- [ ] Tipografia legível em thumbnail (test 100x100px)
- [ ] Nenhuma imagem cartoonish (deve ser cinematográfica)
- [ ] Luz divina/glow presente (não obrigatório mas recomendado)
- [ ] Logo "O Bom Pastor" consistente
- [ ] Sem poluição visual (máximo 3 cores primárias + branco)
- [ ] Texto com sombra ou contorno (legível em todo fundo)
- [ ] Ícones/emojis usados de forma estratégica
- [ ] Proporções corretas (não distorcido)
- [ ] Marca d'água ou assinatura clara

## Referências de Inspiração

**Cores:**
- Sunset/Golden Hour palettes
- Old Masters paintings (ouro, marrom, escuro)
- Religious art (não icônico, mas inspirado)

**Tipografia:**
- Montserrat: Moderna mas forte
- Open Sans: Limpa e acessível

**Imagens:**
- Pastor com rebanho (Salmos 23)
- Luz divina/raios de sol
- Paisagens épicas (montanha, mar calmo)
- Mãos em oração
- Pessoas em paz/contemplação

**Evitar:**
- ❌ Clipart ou illustrations genéricas
- ❌ Muitas cores (máximo 5)
- ❌ Efeitos brilhantes demais
- ❌ Fontes muito modernas/gaming
- ❌ Imagens de baixa qualidade

---

**Mantenha essa consistência em TODOS os assets. Qualidade visual consistente constrói reconhecimento de marca.**
