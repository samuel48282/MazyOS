## ESTRATÉGIA POR TIPO

### 📱 SHORTS (60-90s) — Frequência: 5/semana

**Temas que viralizam:**
- Semana 1: Saúde (problema específico)
- Semana 2: Fé/Motivação (reflexão)
- Semana 3: Saúde (solução)
- Semana 4: Motivação (transformação)

**Roteiro padrão:**
```
[0-5s] Gancho visual + voz emocional
[5-30s] Problema específico
[30-50s] Solução/Reflexão
[50-60s] CTA (salvar, compartilhar, comentar)
```

### 📹 YOUTUBE (12-20 min) — Frequência: 2/semana

**Rotação:**
- 1x Saúde Natural (rotina, receita, propriedades)
- 1x Fé/Motivação (reflexão profunda, histórias)

**Estrutura:**
```
[0-1m] Intro emocional
[1-15m] Conteúdo prático/profundo
[15-19m] Resumo + CTA
[19-20m] Próximo vídeo + inscrição
```

### 🎬 REELS (60s) — Frequência: 2/semana

**Fórmulas:**
- Receita visual (ingredientes + benefício)
- Meditação/Yoga (início + fim + paz)
- Depoimento rápido (transformação)
